# 🚀 Supply Chain Analytics using AI Tools (Quadratic + n8n + Supabase)

## 📌 Project Overview
This project focuses on analyzing and improving supply chain performance using modern AI-powered tools and workflow automation.

An end-to-end pipeline was built to ingest, process, and analyze order-level data, enabling the identification of delivery inefficiencies, revenue leakage, and customer service gaps.

---

## 🧠 Problem Statement
Businesses often face challenges such as:
- Delayed deliveries
- Incomplete shipments
- Revenue loss due to unfulfilled orders
- Lack of real-time visibility into supply chain KPIs

This project addresses these issues by building a data-driven system to monitor and analyze supply chain performance.

---

## 🛠️ Tech Stack
- **Quadratic** – AI-powered spreadsheet for analysis
- **n8n** – Workflow automation & data pipeline orchestration
- **Supabase (PostgreSQL)** – Cloud database for structured storage
- **Gmail API (OAuth 2.0)** – Automated communication integration
- **Python** – Data cleaning, transformation, and KPI computation

---

## ⚙️ End-to-End Architecture
1. Automated data ingestion using n8n workflows  
2. Data stored in Supabase (PostgreSQL database)  
3. Data cleaning and transformation using Python  
4. Analytical processing using Quadratic AI  
5. KPI computation and business insights generation  

---

## 📊 Key KPIs Implemented
- **Line Fill Rate** – % of order lines fulfilled  
- **Volume Fill Rate** – % of total quantity delivered  
- **On-Time Delivery %** – Orders delivered within deadline  
- **In-Full Delivery %** – Orders delivered completely  
- **OTIF % (On-Time In-Full)** – End-to-end delivery reliability  

---

## 🔍 Key Business Insights
- Identified **revenue loss due to undelivered orders**
- Detected **customers with high OTIF failures**
- Highlighted **product categories with low fulfillment rates**
- Measured **average delivery delays impacting service levels**
- Exposed **supply chain bottlenecks affecting order completion**
